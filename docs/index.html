POS License Header Helper

POS License Header Helper is a Chrome and Edge extension for internal KPOS deployments. It helps authorized client devices add required identity headers to KPOS API requests.

POS License Header Helper 是用于公司内部 KPOS 系统的浏览器扩展，帮助 POS、电子菜单和自助点餐终端完成客户端授权请求头注入。
