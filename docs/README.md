# GitHub Pages

This folder contains the static GitHub Pages site for POS License Header Helper.

## Suggested Chrome Web Store Privacy Policy URL

After enabling GitHub Pages with source `main` branch and `/docs` folder, use:

```text
https://<github-user-or-org>.github.io/<repository-name>/privacy/pos-license-header-helper/
```

If this repository is published as a user or organization site, use:

```text
https://<github-user-or-org>.github.io/privacy/pos-license-header-helper/
```

## GitHub Pages Setup

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. In `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/docs`.
6. Save and wait for GitHub Pages to publish.
